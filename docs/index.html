Powerful Features

Built with Rust and Tauri for maximum performance and minimal resource usage.

Lightning Fast

Speak up to 4x faster than typing. Kalam captures your voice and injects the text instantly into whatever application you're using.

Privacy First

Your audio is never stored to disk. Choose between secure cloud processing (zero retention) or completely offline local models.

Dual-Engine STT

Use Groq's lightning-fast Whisper API for cloud processing, or switch to SenseVoice/Whisper.cpp for entirely on-device transcription.

Global Hotkeys

Activate dictation from anywhere with a simple keyboard shortcut (Ctrl+Win / Ctrl+Cmd). No need to switch windows.

Built-in Command Center

Smart Notes

Capture fleeting thoughts instantly. Tag, pin, and organize your voice notes in a dedicated, searchable workspace.

Task Management

Brain dump your to-dos with your voice. Kalam organizes them into actionable checklists you can manage later.

Voice Reminders

Set reminders completely hands-free. Never forget a follow-up, meeting, or random idea again.

Text Snippets

Create voice shortcuts for things you type often. Say a short trigger phrase to instantly paste long templates or links.

Frequently asked questions

Is Kalam free?
Yes. Kalam is open source and free. You can use cloud transcription with your own API key (Groq has a free tier) or run fully offline with local models—no subscription.
Is Kalam like Wispr Flow or Whisperflow?
Yes—same idea (voice typing in any app). Kalam is free, open source, and runs on Windows, Mac, and Linux (desktop only); Wispr Flow is paid and also on mobile.
What platforms does Kalam support?
Kalam runs on Windows, macOS, and Linux (desktop only). We are not building for Android or iOS.
Does Kalam work offline?
Yes. You can use a local STT model (SenseVoice or Whisper) so no audio leaves your device. No API key required for offline mode.
Do I need an API key?
Only for cloud transcription (e.g. Groq). Sign up at Groq Console, get a free key, and add it in Settings → STT Provider. For local mode, no key is needed.
Is my audio stored or sent somewhere?
Audio is not stored to disk (in-memory only). In cloud mode it’s sent to the provider over TLS and not retained. In local mode, nothing leaves your machine.
How do I start dictating?
Press Ctrl+Win (Windows) or Ctrl+Cmd (macOS), hold while you speak, then release. Text is typed into the app that had focus when you pressed the key.
