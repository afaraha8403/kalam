Before you install

Kalam is distributed outside app stores. The first time you run it, your OS may show a security warning — that is expected for open-source builds that are not store-notarized.

What to expect

Permissions

Kalam only asks for what it needs to dictate and respect your privacy settings.

Microphone

Required for voice dictation. Audio is processed locally or sent only to the STT provider you configure.

Accessibility / input

Needed to insert text into other apps and to detect the active window for Smart Privacy mode.

Notifications

Optional — for reminders and due dates from the built-in workspace.

First launch

Security prompts

We matched a guide to your system. Expand for step-by-step help.

Windows — SmartScreen

SmartScreen may show “Windows protected your PC.” Choose More info, then Run anyway. This appears because the installer is not signed with an EV certificate — the app is open-source and safe to use.

macOS — Gatekeeper

If you see “from an unidentified developer,” open System Settings → Privacy & Security, scroll down, and click Open Anyway. Or right-click the app and choose Open.

Linux — AppImage / .deb

AppImage: chmod +x Kalam*.AppImage then run. .deb: sudo dpkg -i kalam*.deb. Some distros may need a --no-sandbox flag.

Other / unknown OS

Kalam is a desktop app for Windows, macOS, and Linux. If detection missed your platform, choose an installer from the list below or use GitHub Releases.
