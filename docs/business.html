Sponsors & commercial

Use Kalam in your business, get a commercial license, or support the project.

Commercial license

Kalam is free for noncommercial use (personal, educational, nonprofit) under the MIT license. Commercial use — including use by a business, resale, or integration into a for‑profit product — requires a separate written license from the project owner.

If your company wants to use Kalam commercially, we’re happy to discuss terms. Commercial licenses help sustain development and keep the open-source version free for everyone.

See the full terms in the LICENSE file in the repo.

Request commercial license Or open a GitHub issue if you prefer.

Sponsor the project

Kalam is built and maintained as open source. Sponsors help keep the project sustainable, fund infrastructure, and support new features — so everyone can keep using free, privacy-friendly voice typing.

You can sponsor Kalam via GitHub Sponsors — the sponsor button is live on this repo and the maintainer's profile — or get in touch to discuss sponsorship (one-time or ongoing). We’re grateful for every contribution.
